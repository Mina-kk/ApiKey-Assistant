关于应用

          
        

        

          

            ApiKey助手
            当前版本：3.0.71
          

          

            
软件功能

            
    
              
  • 集中保存和管理多个 AI 服务渠道、Base URL 与 API Key。
    • 
              
  • 支持 OpenAI、Claude、Gemini、DeepSeek、硅基流动、OpenRouter 等多种兼容渠道。
    • 
              
  • 可所属分组管理、搜索、排序、批量启用/禁用、批量获取模型。
    • 
              
  • 支持已配模型管理、模型重定向、NewAPI 工具。
    • 
              
  • 提供模型连通性测试、导入/导出 JSON、复制 Key/配置等常用操作。
    • 
            

          

          

            
简单用法

            
    
              
  1. 点击右下角 + 新建渠道。
    2. 
              
  2. 填写名称、服务类型、Base URL 和 API Key。
    3. 
              
  3. 点击“获取模型”拉取已配模型，也可手动添加模型。
    4. 
              
  4. 在卡片中复制 Key、复制配置，或进入“模型诊断”验证可用性。
    5. 
              
  5. 用左侧所属分组管理抽屉整理渠道；更多菜单里可导入、导出、批量管理。
    6.
